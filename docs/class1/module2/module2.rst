Lab 2 - Speed of Automation and Basic BIG-IP Tasks
==================================================

In this lab you will leverage Ansible to deploy a new application
and automate configuring F5 app delivery services.  This demonstrates how automation
can increase application deployment process for many companies.

In addition this module will include exercises to do basic administrative tasks on the BIG-IP.

The following recipes target a common subset of work that people
typically undertake when configuring BIG-IP devices. Among the
various recipes included in this chapter are means to create pools
and virtual servers.
